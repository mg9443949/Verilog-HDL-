# Verilog HDL Projects

A collection of Verilog HDL (Hardware Description Language) implementations ranging from basic logic gates and combinational circuits to sequential logic components like flip-flops and counters.

## 📂 Repository Structure

This repository contains functional Verilog modules for the following digital components:

### Combinational Logic
* **Multiplexers:** 2:1 Mux (using if-else) and 4:1 Mux.
* **Arithmetic:** Carry Ripple Adder.

### Sequential Logic
* **Flip-Flops:** * SR Flip-Flop (implemented via `case` and `if-else` statements).
    * JK Flip-Flop (implemented via `case` and `if-elseif`).
    * D Flip-Flop and T Flip-Flop.
* **Counters:** Up Counter, Down Counter, and a bidirectional Up-Down Counter.
* **Miscellaneous:** Duty Cycle modulation module.

## 🚀 Getting Started

### Prerequisites
To simulate these files, you will need a Verilog simulator such as:
* [Icarus Verilog](http://iverilog.icarus.com/)
* [ModelSim](https://www.intel.com/content/www/us/en/software/programmable/quartus-prime/model-tech-codesign/modelsim.html)
* [Vivado Design Suite](https://www.xilinx.com/products/design-tools/vivado.html)
